## Timestamp to string.

A small util for converting a timestamp to a string.
- "XX seconds"
- "XX minutes"
- "XX hours"
- "XX weeks"
- "XX months"
- "XX years"

## Installation

**Installation**

```
npm install
```

## Test

**Run tests**

```
npm test
```

## Usage

```
import timestampToString from '../src/time';

timestampToString(milliseconds)
```
